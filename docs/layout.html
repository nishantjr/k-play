Custom comments

Productions for the #Layout sort are used to describe tokens that are considered “whitespace”. For example, below, we use it to define lines begining with ; (semicolon) as comments.

module LAYOUT
  syntax #Layout ::= r"(;[^\\n\\r]*)"
                   | r"([\\ \\n\\r\\t])" // Whitespace
                   
  syntax KItem ::= "foo"
endmodule
foo ; buzz
; bar
<k>
  foo
</k>
